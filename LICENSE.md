```bash

```

#### Issue 2: Scripts Not Executing

**Symptoms:** Scripts load but don't run when pressing execute

**Troubleshooting Steps:**
1. Verify the supported title is running
2. Check if auto-attach is enabled in settings
3. Try manual attach from the menu
4. Clear script cache: Settings → Advanced → Clear Cache
5. Update to latest version

```bash
# Manual attach command (advanced users)
attach_process([game_name])
execute_script([script_content])
```

#### Issue 3: High CPU/Memory Usage

**Solutions:**
- Reduce `max_memory` in config file
- Disable script caching
- Close unused tabs
- Update graphics drivers
- Reduce injection frequency

#### Issue 4: Antivirus Blocking Installation

**Resolution:**
1. Create exclusion for application folder
2. Windows Defender: Settings → Virus & threat protection → Exclusions → Add folder
3. Add both the installation directory and temp folder
4. Whitelist the executable file specifically

#### Issue 5: Mobile Version Crashing

**Android Fixes:**
- Clear app cache: Settings → Apps → [App Name] → Clear Cache
- Ensure overlay permission is granted
- Check for Android system updates
- Reinstall using official APK source

**iOS Fixes:**
- Delete and reinstall app
- Revoke and re-trust certificate
- Check iOS version compatibility
- Free up storage space (requires 500MB+)
